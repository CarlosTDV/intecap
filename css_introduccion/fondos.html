Fondos CSS

Las propiedades de fondo de CSS se utilizan para agregar efectos de fondo a los elementos.

'background-color':

La propiedad 'background-color' se utiliza para establecer el color de fondo de un elemento. Puede ser especificado usando nombres de color, valores hexadecimales, valores RGB, entre otros. Ejemplo: 

                section {
                    background-color: #f0f0f0;
                }

'background-image':

La propiedad 'background-image' se utiliza para establecer una imagen de fondo para un elemento. Puede especificar la ruta de la imagen o utilizar 'url()' para incluir la imagen. Ejemplo: 

                section {
                    background-image: url('img/fondo.jpg');
                }

'backround-repeat':

La propiedad 'backround-repeat' se utiliza para especificar si la imagen de fondo debe repetirse en el eje horizontal, vertical o en ambos. Puede tener valores como 'repeat', 'repeat-x', 'repeat-y', o 'no-repeat'. Ejemplo: 

                section {
                    background-repeat: repeat-x;
                }

'background-attachment':

La propiedad 'background-attachment' se utiliza para especificar si la imagen de fondo debe estar fija o desplazarse con el contenido cuando el usuario desplace la página. Puede tener valores como 'scroll' o 'fixed'. Ejemplo: 

                section {
                    background-attachment: fixed;
                }

'background-position':

La propiedad 'background-position' se utiliza para definir la posición inicial de la iagen de fondo. Puede especificar valores como píxeles, porcentajes o palabras clave como 'left', 'center', 'top', 'bottom', etc. Ejemplo: 

                section {
                    background-position: center top;
                }

Propiedad 'opacity'

La propiedad 'opacity' en CSS se utiliza para controlar la opacidad de un elemento, es decir, cuantó se ve a través de él. Puede tomar valores en el rango de 0 a 1, donde 0 representa la completa (sin transparencia). La sintaxis básica es la siguiente: 

                selector {
                    opacity: valor;
                }

'valor' puede contener un número en el rango de 0 a 1. Por ejemplo, '0.5' representa una opacidad del 50%.
