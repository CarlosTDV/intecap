¿Que es CSS?

CSS (Cascading Style Sheets) significa "Hojas de Estilo en Cascada", es un lenguaje de diseño utilizado para describir la presentación de un documento HTML o XML. CSS se centra en la parte visual de una pagina, como el estilo de fuente, colores, margenes, etc. CSS puede utilizarse en un documento separado de HTML, lo que permite usar en varios documentos los recursos de un solo CSS.

Formas de aplicar CSS

1. En línea

Para agregar CSS en línea, se debe escribir dentro de la etiqueta correspondiente.
En el siguiente ejemplo se agregó CSS en la etiqueta p. 

                <p style="color: red; font-size: 20px;">Así se aplica CSS en línea</p>

Así se aplica CSS en línea

2. Interna

Para agregar CSS de forma interna, se debe escribir dentro de la etiqueta head la palabra style.
Luego, colocar la etiqueta a la que se le aplicaran estilos y entre llaves escribir los parametros correspondientes. A continucación se muestra un ejemplo. 

                <!DOCTYPE html>
                <html lang="es">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Interna</title>
                    <style>
                        p{
                            color: red;
                            font-size: 10px;
                        }
                    </style>
                </head>
                <body>
                    <p>Así se aplica CSS de forma interna</p>
                </body>
                </html>

Así se aplica CSS de forma interna

3. Externa

Para agregar CSS de forma externa, se debe crear un archivo con extensión .css.
Luego, escribir la etiqueta a la que se le aplicara los estilos, y escribir los parametros correspondientes. 

                .p_externo{
                    color: red;
                    font-size: 20px;
                }

Ahora dentro del documento html, con la etiqueta link hacer referencia al archivo css.

                <!DOCTYPE html>
                <html lang="es">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Externa</title>
                    <link rel="stylesheet" href="css/estilos.css">
                </head>
                <body>
                    <p class="p_externo">Así se aplica CSS de forma externa</p>
                </body>
                </html>

Así se aplica CSS de forma externa

Comentarios

Los comentarios ayudan al desarrollador y a cualquier persona que quiera leer el código a identificar que es lo que hace una cada sección del código, de esta forma es mas sencillo ubicar una parte especifica y poder modificarla.
Para escribir un comentario, se debe colocar entre estos simbolos: /* */. Los comentarios son únicamente para lectura, asi que no afecta el funcionamiento de la página. 

                
               /*Este es un comentario*/

                .p_externo{
                    color: red;
                    font-size: 20px;
                }
