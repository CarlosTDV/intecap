En CSS, los bordes son parte del modelo de caja y se utilizan para definir la apariencia visual del borde exterior de un elemento. Los bordes pueden tener diferentes estilos, anchos y colores, y se aplican alrededor del contenido y el relleno de un elemento. La propiedad principal para controlar los bordes es 'border'.
La propiedad 'border' en CSS se puede dividir en tres partes:

  1. Ancho del Borde ('border-width'):

    Define el ancho del borde. Puede especificarse en píxeles, em, rem o porcentajes. Ejemplo: 'border-width: 2px'

  2. Estilo del Borde ('border-style'):

    Define el estilo del borde, como sólido, punteado, discontinuo, doble, etc. Ejemplo: 'border-style: dashed;'

  3. Color del Borde ('border-color')

    Define el color del borde. Puede ser un nombre de color, un valor hexadecimal, un valor RGB, etc. Ejemplo: 'border-color: #333;'

Estas tres propiedades pueden combinarse en la propiedad 'border' de una manera más compacta, espeficicando el ancho, el estilo y el color en ese orden. Por ejemplo: 

                border: 2px dashed #be0000;

Además de la propiedad 'border', hay variantes que te permiten establecer cada lado del borde de forma independiente:

  • Borde Superior ('border-top'):

    Permite establecer las propiedades del borde en la parte superior del elemento.

  • Borde Derecho ('border-right'):

    Permite establecer las propiedades del borde en el lado derecho del elemento.

  • Borde Inferior ('border-bottom')

    Permite establecer las propiedades del borde en la parte inferior del elemento

  • Borde Izquierdo ('border-left'):

    Permite establecer las propiedades del borde en el lado izquierdo del elemento.

Ejemplo:

                border-top: 1px solid #ccc;
                border-right: 2px dotted #666;
                border-bottom: 3px double #999;
                border-left: 4px dashed #aaa;

Bordes redondeados

Los bordes tambien tienen una propiedad que permite suavizar las equinas de un elemento, dando un aspecto más estilizado y moderno. Esta característica se logra mediante la propiedad 'border-radius'. Esta propiedad define el radio de las esquinas de un elemento, lo que determina cuánto se redondean.

La propiedad 'border-raidus puede tomar uno o más valores. Aquí tienes algunos ejemplos:

  1. Un solo valor:

    Aplica el mismo radio de borde a todas las esquinas.

                            #bordeuno{
                            border-radius: 10px;
                        }

  2. Dos valores:

    El primer valor se aplica a las esquinas superiores izquierda y derecha, y el segundo valor a las esquinas inferiores izquierda y derecha. 

                            #bordedos{
                            border-radius: 10px 20px;
                        }

  3. Tres valores:

    El primer valor se aplica a la esquina superior izquierda, el segundo valor se aplica a las esquinas superior derecha e inferior izquierda, y el tercer valor se aplica a la esquina inferior derecha. 

                            #bordetres{
                            border-radius: 10px 20px 30px;
                        }

  4. Cuatro valores:

    Cada valor se aplica a una esquina específica en el orden superior izquierda, superior derecha, inferior derecha e inferior izquierda. 

                            #bordecuatro{
                            border-radius: 10px 20px 30px 40px;
                        }

La propiedaed 'border-radius' es versátil y permite experimentar con diferentes valores para lograr el efecto deseado. Los borde redondeados son especialmente útiles para crear interfaces más amigables y atractivas visualemente.
