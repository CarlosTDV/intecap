Las propiedades 'text-shadow' y 'box shadow' son propiedades en CSS utilizadas para agregar sombras a elementos en una página web.

text-shadow

La propiedad 'text-shadow' se utiliza para agregar sombras a texto. Se puede aplicar sombras a letras o palabras para mejorar la legibilidad o para agregar un efecto estilizado. La sintaxis básica es: 

                selector {
                    text-shadow: offset-x offset-y blur-radius color;
                }

  • 'offset-x': Especifica la posicón horzontal de la sombra en relación con el texto. Puede ser un valor positivo (derecha) o negativo (izquierda.)

  • 'offset-y:' Especifica la posición vertical de la sombra en relación con el texto. Puede ser un valor positivo (abajo) o negativo (arriba).

  • 'blur-radius:' Define la cantidad de desenfoque que se aplica a la sombra. Puede ser un valor positivo.

  • 'color': Define el color de la sombra:

Ejemplo:

                h2 {
                    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
                }

Ejemplo de sombra con 'text-shadow'

Este ejemplo crea una sombra para el texto '<h2>' con un desplazamiento de 2 píxeles a la derecha y 2 píxeles hacia abajo, un desenfoque de 4 píxeles y un color semi-transparente negro.

box-shadow

La propiedad 'box-shadow' se utiliza para agregar sombras a cajas (elementos como divs, imágnes, etx.). Se puede usar para resaltar o dar profunidad a los elementos. La sintaxis básica es: 

                selector {
                    box-shadow: offset-x offset-y blur-radius spread-radius color;
                }
  • 'offset-x': Especifica la posición horizontal de la sombra en relación con la caja.
  • 'offset-y': Especifica la posición vertical de la sombra en relación con la caja.
  • 'blur-radius': Define la cantidad de desenfoque que se aplica a la sombra.
  • 'spread-radius': Aumenta o reduce el tamaño de la sombra.
  • 'color': Define el color de la sombra.

Ejemplo:

                div {
                    box-shadow: 4px 4px 8px rgba(0, 0, 0, 0.3);
                }
Ejemplo de sombra con 'box-shadow'

Este ejemplo agrega una sombra a un '<div>' con un desplazamiento de 4 píxeles a la derecha, 4 píxeles hacia abajo, un desenfoque de 8 píxeles y un color semi-transparente negro.
