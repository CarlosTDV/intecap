Estructura de CSS

En CSS, la estructura básica consta de tres partes principales: selector, propiedad y valor.

                selector{propiedad:valor;}

a. Selector

El selector es la parte que llama a los elementos HTML a los que se le aplicarán los estilos. Puede ser directamente el nombre de un elemento HTML, una clase o un ID. Si se coloca el nombre del elemento HTML, los parametros dentro de este se aplicarán a todos los elementos dentro del documento HTML, una clase solo afecta a los elementos con que hagan referencia a esta, mientras que un ID afecta unicamente a al elemento que posea dicha ID.

b. Propiedad

La propiedad define qué caractéristica o aspecto del elemento se va a utilizar. Puede ser cualquier elemento visual, como la fuente del texto, color o tamaño.

c. Valor

El valor es la configuración específica que se asigna a la propiedad. Representa el estilo o apariencia que se le aplicará a elemento seleccionado.

Modelo de cajas en CSS

El modelo de cajas en CSS se refiere a cómo se representa y organiza el contenido de un elemento HTML en términos de cajas rectangulares. Cada elemento HTML se considera una caja, y el modelo de cajas describe cómo se calculan y aplican propiedades como el ancho, alto, márgenes, bordes y rellenos de esa caja. El modelo de caja consta de las siguientes partes principales:

  1. Contenido (Content):
    Es el área donde se muestra el contenido real del elemento, como texto, imágenes, u otros elementos HTML.
  2. Relleno (Padding):
    Es el espacio entre el contenido y el borde de la caja. El relleno proporciona a un espacio adicional dentro de la caja y afecta el tamaño total del elemento.
  3. Borde (Border):
    Es la línea que rodea el contenido y el relleno de la caja. Puede tener diferentes estilos, anchos y colores.
  4. Margen (Margin):
    Es el espacio entre el borde de la caja y los elementos adyacentes. Los márgenes se utilizan para controlar el espacio entre las cajas.

Cuando se aplica estilos CSS a un elemento, se ajustan las propiedades relacionadas con el modelo de caja para controlar su apariencia y diseño en la página web. Por ejemplo, se puede establecer el tamaño del contenido, el color de fono, el ancho del borde y el espacio entre elementos utilizando propiedades como 'widht', 'height', 'padding', 'border' y 'margin'.

Elementos de línea y elementos de bloque

En HTML y CSS, los elementos se dividen en dos categorias principales en función de cómo se comportan y cómo afectan al flujo de contenido en una página web. Estas categorías son elementos de lína (inline elements) y elementos de bloque.

  1. Elementos de Línea (Inline Elements):

    Los elementos de línea se comportan como si estuvieran en una línea de texto. No inician una nueva línea y solo ocupan espacio horizontal necesario. Por lo tanto, múltiples elementos de línea pueden aparecer en la misma línea. Ejemplos comunes de elementos de línea son:

    • '<span>'
    • '<a>' (hipervinculo)
    • '<strong>' (negrita)
    • '<em>' (énfasis o cursiva)
    • '<img>' (imagen)
    • '<br>' (salto de línea)

    Un ejemplo de uso de elementos de línea puede ser:

                            <p>Este es un texto <strong>en negrita </strong> y un <a href="#">hipervinculo</a>.</p>

    Este es un texto en negrita y un hipervínculo.

  2. Elementos de Bloque (Block Elements):

    Los elementos de bloque, por otro lado, comienzan en una nueva línea y ocupan todo el ancho disponible. Cada elemento de bloque crea un "bloque" independiente en la pagina. Ejemplos comunes de elementos de bloque son:

    • '<div>'
    • '<p>' (párrafo)
    • '<h1>' a '<h6>' (encabezados)
    • '<ul>', '<ol>', '<li>' (listas)
    • '<table>' (tabla)
    • '<form>' (formulario)
    Ejemplo de uso de elementos de bloque: 
                            <div>
                            <h2>Este es un encabezado de bloque</h2>
                            <p>Este es un párrafo dentro de un bloque.</p>
                        </div>

Es importante entender la diferencia entre estos dos tipos de elementos, ya que afecta cómo se estructura y presentan los contenidos en un página web. Además, CSS se puede utilizar para cambiar el comportamiento de los elementos, por ejemplo, convirtiendo un elemento de línea en un elemento de bloque o viceversa utilizando la propiedad 'display'.
