Los colores se especifican utilizando nombres de colores predefinidos o valores RGB, HEX, HSL, RGBA, HSLA.

Nombres de colores CSS

En CSS, se puede especificar un color utilizando un nombre de color ya definido:

red orange
brown green
darkcyan blue
purple firebrick

Valores de colores

Los valores pueden alterarse para obtener una mayor variedad de colores, aunque tambien se pueden utilizar para replicar colores ya definidos.

Valores RGB:

rgb(255, 0, 0) rgb(255, 165, 0)
rgb(165, 42, 42) rgb(0, 128, 0)
rgb(0, 139, 139) rgb(0, 0, 255)
rgb(128, 0, 128) rgb(178, 34, 34)

Valores HEX:

#ff0000 #ffa500
#a52a2a #008000
#008b8b #0000ff
#800080 #b22222

Valores HSL:

hsl(0, 100%, 50%) hsl(39, 100%, 50%)
hsl(0, 59%, 41%) hsl(120, 100%, 25%)
hsl(180, 100%, 27%) hsl(240, 100%, 50%)
hsl(300, 100%, 25%) hsl(0, 68%, 42%)
