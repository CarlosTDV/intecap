If, else if y else

Las directivas @if, @else if, y @else en Sass permiten el uso de condicionales para aplicar estilos en función de determinadas condiciones. Estas directivas son útiles para crear hojas de estilo dinámicas y adaptables, lo que puede simplificar el mantenimiento y la personalización del CSS.

Uso de @if, @else if, y @else

Las directivas condicionales en Sass funcionan de manera similar a las condicionales en otros lenguajes de programación. La sintaxis básica es:

                    @if <condición> {
                        // Estilos si la condición es verdadera
                      } @else if <otra condición> {
                        // Estilos si la otra condición es verdadera
                      } @else {
                        // Estilos si ninguna condición anterior es verdadera
                      }

Ejemplo Práctico

Supongamos que queremos cambiar el color de fondo de un botón según el valor de una variable:

                    @mixin button-theme($theme) {
                        padding: 10px 20px;
                        color: white;
                        border: none;
                        border-radius: 5px;
                        cursor: pointer;
                        @if $theme == light {
                          background-color: #3498db; // Azul claro para el tema claro
                        } @else if $theme == dark {
                          background-color: #2c3e50; // Azul oscuro para el tema oscuro
                        } @else {
                          background-color: gray; // Gris si el tema no es ni claro ni oscuro
                        }
                      }
                      
                      .button-theme-1 {
                        @include button-theme(dark);
                      }
                      .button-theme-2 {
                        @include button-theme(light);
                      }
                      .button-theme-3 {
                        @include button-theme(random);
                      }

Con la ayuda de un @mixin es posible pasar parametros, y estos a su vez condicionarlos según nuestras necesidades
