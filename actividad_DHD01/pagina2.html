Mixins

En Sass, los mixins son bloques de código CSS reutilizables que te permiten escribir código más organizado, mantenible y consistente. Se pueden utilizar para agrupar propiedades CSS comunes, definir estilos de base para componentes y crear funciones CSS personalizadas. ¿Cómo funcionan los mixins en Sass? Los mixins en Sass se definen utilizando la directiva @mixin. El nombre del mixin va seguido de llaves que contienen las propiedades CSS que se definen en el mixin. Los mixins se pueden incluir en otros selectores de CSS utilizando la directiva @include. Ejemplo de un mixin en Sass: 

                    @mixin button-primary {
                        background-color: #007bff;
                        color: #fff;
                        border: none;
                        padding: 10px 20px;
                        text-align: center;
                        text-decoration: none;
                        display: inline-block;
                        font-size: 16px;
                        margin: 4px 2px;
                        cursor: pointer;
                      }
                      
                      .button {
                        @include button-primary;
                      }

En este ejemplo, se define un mixin llamado button-primary que define las propiedades CSS para un botón primario. El mixin se incluye luego en el selector .button para aplicar los estilos de botón primario al elementos.

@extends

La directiva @extend en Sass se utiliza para heredar las propiedades CSS de un selector a otro. Esto significa que puedes definir un conjunto de propiedades CSS en un selector y luego aplicar esas mismas propiedades a otros selectores utilizando la directiva @extend. Coloquemos las propiedades del mixin anterior dentro del selector .button: 

                    .button {
                        background-color: #007bff;
                        color: #fff;
                        border: none;
                        padding: 10px 20px;
                        text-align: center;
                        text-decoration: none;
                        display: inline-block;
                        font-size: 16px;
                        margin: 4px 2px;
                        cursor: pointer;
                    }

Ahora heredaremos esas propiedades al selector .button-small con algunos cambios:

                    .button-small {
                        @extend .button;
                        padding: 5px 10px;
                        font-size: 12px;
                    }

Esto es especialmente útil cuando varios elementos son casi identicos y solo se necesita cambiar algunas propiedades.
