Funciones

En Sass, las funciones son bloques de código reutilizables que realizan cálculos y transformaciones y devuelven un valor. Las funciones en Sass permiten encapsular lógica y operaciones complejas que se pueden utilizar en diferentes partes del código, mejorando la modularidad y la mantenibilidad del estilo CSS.

Definir funciones en Sass

Las funciones en Sass se definen utilizando la directiva @function. Esta es la sintaxis básica:

                    @function nombre-funcion($parametro1, $parametro2, ...) {
                        // Código de la función
                        @return valor;
                    }

Ejemplo: Calcular el brillo de un color

Una función útil podría ser calcular el brillo de un color y devolver un color más claro o más oscuro:

                    @function adjust-brightness($color, $amount) {
                        @if $amount > 0 {
                          @return lighten($color, $amount);
                        } @else {
                          @return darken($color, abs($amount));
                        }
                      }
                      
                      // Uso de la función
                      .button-color-1 {
                        background-color: adjust-brightness(#3498db, 20%);
                        border-color: adjust-brightness(#3498db, -10%);
                      }
                      .button-color-2 {
                        background-color: adjust-brightness(#3498db, -10%);
                        border-color: adjust-brightness(#3498db, 20%);
                      }
