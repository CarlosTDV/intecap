For, each, while

Las directivas @for, @each, y @while en Sass permiten realizar iteraciones, lo que facilita la generación dinámica de CSS basado en listas o rangos de valores. Estas directivas son útiles para crear estilos repetitivos de manera eficiente y mantener el código limpio y organizado.

Directiva @for

La directiva @for se utiliza para iterar sobre un rango de números, generando reglas CSS repetitivas con diferentes valores.

                    @for $var from  through  {
                        // Código a ejecutar
                      }

Ejemplo

Crear clases para columnas con anchos porcentuales:

                    @for $i from 1 through 12 {
                        .col-#{$i} {
                          width: 100% / 12 * $i;
                          border: 1px solid black;
                        }
                      }

Directiva @each

La directiva @each se utiliza para iterar sobre listas o mapas, aplicando estilos a cada elemento.

                    //Para Listas
                    @each $item in $list {
                        // Código a ejecutar
                      }
                      //Para Mapas
                      @each $key, $value in $map {
                        // Código a ejecutar
                      }

Ejemplo con Listas

Crear clases de colores basados en una lista:

                    $colors: red, green, blue;

                        @each $color in $colors {
                        .text-#{$color} {
                            color: $color;
                        }
                    }

Crear clases de temas basados en un mapa:

                    $themes: (

Directiva @while

La directiva @while se utiliza para ejecutar un bloque de código repetidamente mientras se cumpla una condición.

                    @while 'condicion' {
                        // Código a ejecutar
                      }

Ejemplo

Crear clases para elementos decrecientes:

                    $i: 6;

                    @while $i > 0 {
                        .item-#{$i} {
                            width: 10px * $i;
                            border: 1px solid black;
                        }
                    $i: $i - 1;
                    }
