Conceptos básicos

Regresar

  • Sitio web

    Un sitio web es un conjunto de paginas enlazadas entre si. Estas se encuentran alojadas en un servidor que es accesible para cualquier persona que cuente con acceso a internet.

  • Página web

    Una página web es un documento digital adaptado a la world wide web (www) y es capaz de contener texto y contenido multimedia. Es accesible desde un navegador web para cualquier persona.

  • Hipervinculo

    Un hipervinculo es un enlace que hace referencia a un recurso digital, ya sea un documento, imagen, un video o alguna pagina.

  • URL

    URL significa Uniform Resource Locator (Localizador de Recursos Uniforme). Es una dirección única y especifica dada a un recurso en la web, ya sea una pagina, un documento o incluso una imagen. Todos estos recursos estan enlazados por una URL para que los navegadores puedan localizarlos.

  • HTML

    HTML (Hypertext Markup Language) es un conjunto de etiquetas que conforman los elementos de una página web, como textos, imagenes, listas, videos.

  • CSS

    CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML. Se encarga de darle formla, color, posición y otras caracteristicas a un documento web.

  • JS (JavaScript)

    JavaScript es un lenguaje de programación que permite implementar funciones complejas en páginas web. permite la actualización de contenido, animaciones.

  • DNS

    DNS es un sistema de nombres de dominio. Las personas normalmente acceden a una pagina a tráves de un nombre de dominio como google.com o wikipedia.org, y el el DNS se encarga de traducir el nombre de domino a una dirección ip para que los navegadores puedan cargar la página.

  • IP

    Una IP (Internet Protocol) es una etiqueta numérica unica que identifica un dispositivo en internet o una red local. Las direcciones IP se identifican con cuatro números, ejemplo, 192.168.1.38. Cada número puede variar de 0 a 255.

  • Dominio Web

    Un dominio web es un espacio que contiene todo tipo de rescursos en internet. Estos estan asociados a personas y organizaciones que los hayan registrado, como google.com o microsoft.com.

  • Cliente-Servidor

    Es un modelos de comunicación que permite la distribución de tareas dentro de una red de ordenadores. Un servidor puede ser un programa informatico que recibe peticiones del cliente, las procesa y da una respuesta a la solicitud.

  • FTP

    FTP (File Transfer Protocol) es un protocolo que se utiliza para transferir todo tipo de archivos entre equipos conectados a una red. Es el medio estándar para gestionar el contenido alojado en un servidor web.

  • HTTP

    HTTP (Hypertext Transfer Protocol) es un protocolo de transferencia de hipertexto y se utliliza para cargar páginas web emdiante enlaces de hipertexto. Esta diseñado para transferir información entre dispositivos conectados a la red.

  • HTTPS

    HTTPS (Hypertext Transfer Protocol Secure) es la versión segura de HTTP. Este está encriptado para aumentar la seguridad de las transferencias de datos. Esto es especialmente importante cuando los usuarios transmite datos confidenciales.

  • Frontend

    Frontend es la parte visual de una aplicación, todo aquello con lo que los usuarios pueden ver e interactuar, como los textos, imagenes, botones, casillas, menus de navegación.

  • Backend

    Backend es la infraestructura de la aplicación, lo que hace que está funcione, almacena datos y los procesa para los usuarios. Cuando el usuario interactua con el front end, este envia una solicitud al back end en formato HTTP, procesa la solicitud y devuelve una respuesta.

  • Hosting

    Un hosting es un servicio online que hace que se pueda acceder a un sitio web en internet. Los proveedores proporcionan un espacio en un servidor que almacena todos los archivos y datos de un sitio web y hace que funciones correctamente.

  • Hosting Privado Virtual (VPS)

    Un VPS (Virtual Private Server) es un espacio en un servidor de menor escala y más barato que alquilar un servidro hosting completo. Es normalmente usado por personas que no necesitas los recursos de un servidor dedicado.

  • Hosting en la nube (Cloud Hosting)

    El Cloud Hosting funciona mediante la virtualización. Es una tecnologiá que divide un servidor físico en varias maquinas virtuales llamadas serviodores en la nube. Luego, estas se conectan para crear una red única para alojar un sitio web.

  • W3C

    W3C (World Wide Web Consortium) es un consocio internaciona que genera recomendaciones y estándares que asegura el crecimeiento de la World Wide Web (WWW) a largo plazo.
