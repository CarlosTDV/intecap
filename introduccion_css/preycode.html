Ejemplo para incluir css en un documento

1. Incluir css dentro del documento html

El codigo CSS debe escribirse dentro de la etiqueta style dentro del head 

            <!DOCTYPE html>
            <html lang="es">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
                <style>
                    code pre{
                        width: 100%;
                        background-color: antiquewhite;
                        border: 1px solid black;
                    }
                </style>
            </head>

2. En Linea

Se puede escribir como atributo de la etiqueta style
