CSS (Cascading Style Sheets) significa "Hojas de Estilo en Cascada", es un lenguaje de diseño utilizado para describir la presentación de un documento HTML o XML. CSS se centra en la parte visual de una pagina, como el estilo de fuente, colores, margenes, etc. CSS puede utilizarse en un documento separado de HTML, lo que permite usar en varios documentos los recursos de un solo CSS.
